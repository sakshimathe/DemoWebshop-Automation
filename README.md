# Demo Web Shop Automation Testing

## Project Overview

This project automates the Demo Web Shop website using Selenium WebDriver. The automation framework is designed to validate key functionalities of the application and improve testing efficiency.

## Technologies Used

* Java
* Selenium WebDriver
* TestNG
* Maven
* Git
* GitHub

## Features Automated

* User Registration
* Login
* Product Search
* Add to Cart
* Shopping Cart
* Checkout
* Logout

## Project Structure

* Test Scripts
* Page Objects
* Test Data
* Screenshots
* Reports

## How to Run the Project

1. Clone the repository.
2. Open the project in Eclipse or IntelliJ IDEA.
3. Install Maven dependencies.
4. Run the TestNG test suite.
